Module: IOT-Tree Edge
==



This module is supported by the open-source project: [IOTTree AI Edge][iot-tree-edige], which is used to support Edge-related functions in message flow, such as visual recognition.

The internal configuration can include cameras supported by IOT-Tree AI Edge, including Edge runtime environment local cameras and IP cameras.
